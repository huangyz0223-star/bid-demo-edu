## 一、项目概述

**项目名称**：BID_demo - 商业模式教学小组研究Agent系统

**背景**：大学内商业模式教学社团，主要活动包括讲座和小组课题研究。本项目聚焦小组研究模块的AI赋能。

**核心定位**：AI Project Coach / AI Discussion Facilitator

---

## 二、问题与目标

### 核心问题
小组讨论中存在两类失衡：
- 高参与/低能力：表达积极但逻辑弱
- 低参与/高能力（I型）：有深度但不愿发言

### 核心目标
**不是"让大家多说"，而是让每个人在最合适的位置参与，产生有效输出。**

---

## 三、产品架构

### 三层记忆架构
- L1: 教师端（课程总览）
- L2: 小组端（项目记忆）
- L3: 用户端（个人成长）

### 多Agent设计（长期目标）
| 角色 | 职责 |
|------|------|
| 主控Agent | 对外交互、意图识别 |
| 研究员 | 调研、分析 |
| 设计师 | 产品设计 |
| 会议主持 | 会议引导 |
| 评估师 | 发言评估 |

---

## 四、核心功能

| 功能 | 说明 |
|------|------|
| 初始化 | 课题+成员输入，生成Agent人设 |
| Skills | 调研/设计/分工/会议/汇报/进度 |
| 会议引导 | 会前→会中→会后 |
| 角色分配 | 分析者/表达者/调研者/质疑者 |
| 成长追踪 | 发言质量、能力雷达 |
