## 一、项目概述

### 1.1 目标
在**3天内**完成一个可演示的最小版本Demo，用于向老师展示"AI Agent赋能小组研究"的思路和可能性。

### 1.2 核心功能
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 小组初始化 | 输入课题+成员，生成Agent人设 | P0 |
| 基础对话 | 与Agent多轮交互，支持Skills路由 | P0 |
| 自动Memory | 对话后更新项目状态 | P0 |
| 分工规划 | 根据成员生成任务分配 | P0 |
| 会议引导 | 会前议程+会中引导+会后总结 | P1 |
| 进度查询 | 查看当前项目状态 | P1 |

### 1.3 演示场景规划

**演示路径（约10分钟）**：
```
1. 打开应用，展示登录后的主界面（2分钟）
   ↓
2. 演示初始化：输入课题"新能源汽车校园共享充电桩"，
   添加3-4个成员（姓名、专业、兴趣）（3分钟）
   ↓
3. 演示对话：问Agent"帮我们分一下工"（1分钟）
   ↓
4. 演示会议：开启会议，生成议程（2分钟）
   ↓
5. 演示Memory：查看自动更新的项目状态（2分钟）
```

# app/utils/agent.py
class ProjectAgent:
    def __init__(self, project_id: str):
        self.project_id = project_id
        self.memory = MemoryManager(project_id)
        self.system_prompt = self._build_system_prompt()
    
    def chat(self, user_message: str) -> str:
        # 1. 识别Intent
        intent = self._recognize_intent(user_message)
        
        # 2. 路由到对应Skill
        if intent == "division":
            return self._handle_division(user_message)
        elif intent == "research":
            return self._handle_research(user_message)
        # ... 其他Skills
        
        # 3. 默认对话
        return self._default_response(user_message)
```

# app/utils/agent.py
def _auto_update_memory(self, user_message: str, response: str):
    """对话后自动更新Memory"""
    
    # 每5轮对话进行一次总结
    if len(self.messages) % 5 == 0:
        summary_prompt = f"""根据以下对话，总结项目当前状态：
        
用户：{user_message}
AI：{response}

请提取：
1. 项目是否有新进展？
2. 是否有新成果？
3. 是否有新的待办事项？
4. 是否有需要更新的进度信息？

请以JSON格式输出。"""
        
        summary = self.openai_client.chat(summary_prompt)
        # 解析并更新Memory
        self.memory.update_from_summary(summary)
```

## 八、风险预案

| 风险 | 表现 | 应急预案 |
|------|------|----------|
| API调用失败 | 页面卡住无响应 | 添加超时处理，显示友好错误信息 |
| 部署失败 | 无法获得公网链接 | 使用ngrok内网穿透，或本地录屏 |
| 网络不稳定 | 演示时断线 | 准备本地版本，切换到本地演示 |
| 演示时间不够 | 讲不完 | 准备"快速演示"路径，跳过部分功能 |
| API Key失效 | 无法调用GPT | 准备fallback回复："API暂时不可用" |
