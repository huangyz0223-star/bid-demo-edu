**Agent核心逻辑**：

```python
# agent.py 核心结构
class ProjectAgent:
    def __init__(self, project_id):
        self.memory = MemoryManager(project_id)
        self.system_prompt = self._build_prompt()
    
    def chat(self, message: str) -> str:
        # 1. Intent识别
        intent = self._recognize_intent(message)
        
        # 2. Skills路由
        if intent == "division":
            return self._handle_division()
        elif intent == "research":
            return self._handle_research()
        elif intent == "meeting":
            return self._handle_meeting_start()
        elif intent == "progress":
            return self._handle_progress()
        
        # 3. 默认对话
        return self._default_response(message)
```

**Intent识别逻辑**：

| Intent | 关键词 | Skill |
|--------|--------|-------|
| division | "分工"、"分一下"、"任务分配" | 分工规划 |
| research | "调研"、"了解一下"、"市场" | 调研助手 |
| meeting | "开会"、"会议"、"讨论" | 会议主持 |
| progress | "进度"、"看看状态" | 进度追踪 |
| design | "设计"、"方案"、"产品" | 产品设计 |
| report | "汇报"、"PPT"、"演讲" | 汇报准备 |

```

    ↓

    ↓
显示议程，用户确认
    ↓
进入会议模式（对话界面+发言记录）
    ↓
用户发言 → Agent记录+反馈
    ↓
用户点击"结束会议"
```
